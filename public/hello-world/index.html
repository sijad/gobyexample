Go by Example: Hello World

اولین برنامه جمله تاریخی “سلام دنیا” را چاپ می‌کند. کل برنامه همین چند خط هست.

 
package main

import "fmt"

func main() {
    fmt.Println("سلام دنیا")
}

کد را داخل فایل hello-world.go قرار دهید و برای اجرای برنامه از دستور go run استفاده نمایید.

 
$ go run hello-world.go
hello world

مواقعی که نیاز به خروجی باینری برنامه داریم می‌توان از دستور go build استفاده کرد.

 
$ go build hello-world.go
$ ls
hello-world	hello-world.go

و سپس فایل باینری را اجرا نماییم.

 
$ ./hello-world
hello world

حالا که فهمیدیم چگونه کد را اجرا و برنامه را بسازیم اجازه دهید تا چیزهایی بیشتری یاد بگیریم.
